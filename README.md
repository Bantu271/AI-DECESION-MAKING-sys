# NeuroCore AI — Decision Intelligence Platform

## Overview

NeuroCore AI is a futuristic AI/ML Decision Intelligence Dashboard designed to simulate enterprise-grade machine learning workflows using modern frontend technologies.

The platform provides an immersive cyberpunk-inspired interface with AI-driven analytics, dataset management, training simulation, visualization systems, AutoML workflows, and an intelligent assistant.

* Real-time KPI analytics
* Accuracy tracking
* GPU utilization simulation
* AI confidence monitoring

* CSV dataset upload
